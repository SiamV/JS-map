Method map for array.

map - проходит по всем элементам массива и применяет свою функцию callback, записывая их в новый объект

Create users array by id and firstName Example: users.map((item,index,array) => ` ${item.id}:${item.first_name}`)
Create array by users email Example: users.map((item) => ` ${item.email}`)
